# MUSE_AM1_JGR_2025_2026
Jordi González de Regàs.
Ejercicios de programación de la asignatura de Ampliación de Matemáticas 1 (AM1) del Máster Universitario en Sistemas Espaciales (MUSE).


## Contenidos

Milestone 1: Integración de órbitas sin funciones.
Milestone 2: Prototipos para integrar órbitas con funciones.
Milestone 3: Estimación de errores en soluciones numéricas.
Milestone 4: Problemas lineales: Regiones de estabilidad absoluta.
Milestone 5: Problema de los N cuerpos.
Milestone 6: Puntos de Lagrange y estabilidad.
Milestone 7: Explorador de órbitas alrededor de los puntos de Lagrange en el CRTBP.
